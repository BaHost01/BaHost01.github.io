Guia de integração

Passos rápidos para usar a biblioteca em executor ou LocalScript.

1

Carregue o script

Importe o arquivo UILibrary.luau no seu projeto ou loader.

2

Crie sua janela

Defina título, tamanho, ícone do Toggle Ball e tabs principais para organização.

3

Registre elementos

Adicione toggles, sliders, dropdowns e callbacks via Utility.SafeCall.

Exemplo mínimo

local lib = loadstring(game:HttpGet(".../UILibrary.luau"))()
local win = lib:CreateWindow({
  Name = "Demo",
  ToggleIcon = "rbxassetid://525701437"
})

win:Notify({
  Title = "XMCLib",
  Message = "UI carregada com sucesso",
  Corner = "TopRight"
})
