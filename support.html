Perguntas frequentes

Perguntas comuns para quem está migrando para a base 2.4.0.

O Toggle Ball funciona em touch?

Sim. O changelog especifica suporte completo a touch + mouse para arrastar e alternar janelas.

Como evitar memory leak em elementos dinâmicos?

Garanta que cada elemento guarde conexões em _connections e chame o destroy para executar DisconnectAll.

Por que o ZIndex foi refeito?

Para parar conflitos com outras bibliotecas UI. Cada janela recebe um tier base e elementos derivados usam offsets previsíveis.
