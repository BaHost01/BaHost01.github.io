XMCLib Extended Notes

Focused reference built from the current UILibrary.luau internals.

Core Services

Uses Players, TweenService, UserInputService, HttpService and TextService to handle UI, callbacks and sizing behavior.

CustomSignal Registry

Supports newsignal/newcall, firesignal/runsignal, and onsignal/onsignalevent aliases.

Utility Layer

Tween, SafeCall, Connect, DisconnectAll, Clamp, Round and helper constructors for UI objects.

ZIndex strategy

Each new window claims a tier (_TIER_SIZE = 20) and notifications use a high base (_NOTIF_BASE = 600).

Practical recommendation

When creating many floating elements, always call destroy routines that use Utility.DisconnectAll to avoid stale listeners and input duplication.
