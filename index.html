Este site agora documenta a XMCLib UI Library do arquivo UILibrary.luau, com foco nas novidades reais da versão 2.4.0: Toggle Ball, notificações avançadas, layout compacto e correções de memory leak.

Novo

Sistema de notificação avançado

Agora aceita aliases de texto (Label, Labels, Text, Message), corners customizáveis e ações com callback.

Upgrade

Compatibilidade mobile real

Slider, Dropdown e Drag reconhecem Enum.UserInputType.Touch além de mouse, melhorando usabilidade em telas pequenas.

Fix

Conexões globais limpas

As conexões são armazenadas em element._connections e removidas com Disconnect() para reduzir leaks.
